# MEDIC DARIYA

Преміальний bilingual (UA/EN) сайт-маркетплейс медичного одягу зі США.

## Стек
- HTML5 + Tailwind CSS (CDN, `darkMode: 'class'`)
- Vanilla JS (SPA-навігація, i18n, стан у `localStorage`)
- Lucide Icons
- Дизайн: Apple HIG / iOS 17

## Запуск
Просто відкрити `index.html` у браузері, або:
```bash
python3 -m http.server 5173
```

## Функції
- Sticky glassmorphism-хедер із перемикачами теми та мови (UA/EN)
- Hero-секція зі слоганом, філософією бренду та футером із соцмережами
- Реєстрація (валідація email/пароля) — доступ до магазину лише після реєстрації
- Каталог з 4 товарами: розмір, степпер кількості, кнопка «Купити»
- Плаваюча кошик-«пігулка» та iOS Action Sheet з Telegram / Instagram кнопками
